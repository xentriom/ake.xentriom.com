## About command

Project links, bot details, and a quick way to orient new users.

## Usage

```text
/about
```

## Parameters

This command has no parameters.

## What It Does

- Shows the bot identity, current version, and short project summary.
- Returns quick links for the website, invite flow, support server, and GitHub repository.
- Works as the fastest in-Discord entry point for new users.

## Additional Notes

- This command does not require a linked account.
