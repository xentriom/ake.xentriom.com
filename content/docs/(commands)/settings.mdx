## Settings command

Manage your Endvoyant profile, privacy settings, and linked accounts.

## Usage

```text
/settings
```

## Parameters

This command has no parameters.

## What You Can Change

- Your profile visibility
- Your preferred language
- Your DM notification preference
- Each linked account's visibility
- Each linked account's auto check-in setting
- Each linked account's auto code redemption setting
- Which linked account is your primary one

## What It Does

- Opens your personal Endvoyant settings panel.
- Lets you edit profile visibility, language, and DM notification preferences.
- Lists every linked account and provides controls for visibility, auto check-in, auto code redemption, primary selection, and account removal.

## Additional Notes

- The profile editor controls three stored user settings: profile visibility, language, and DM notifications.
- The account editor controls three stored account settings: account visibility, auto check-in, and auto code redemption.
- You can also mark a linked account as primary or remove it entirely from this command.
- The language picker uses Endvoyant's built-in supported language list.
- Changes made here affect commands such as [`/profile`](/docs/profile), [`/development`](/docs/development), [`/exploration`](/docs/exploration), and scheduled attendance behavior.
