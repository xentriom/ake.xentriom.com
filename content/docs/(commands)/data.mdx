## Data command

Export the data Endvoyant currently stores for your Discord user.

## Usage

```text
/data
```

## What It Does

- Generates a ZIP export of the data Endvoyant stores for your Discord user.
- Includes `user.json`, plus account, event, and attempted-code files when available.
- Sends the archive back to you as an attachment.

## Additional Notes

- This is the primary user-facing data export command.
- If you have not linked an account yet, Endvoyant asks you to use [`/add account`](/docs/add).
