## Events command

Browse the current in-game bulletin feed for events, updates, and news.

## Usage

```text
/events
/events name:<event>
```

## Parameters

- `name` (optional)
  Jumps directly to a specific event from autocomplete instead of opening the full catalog first.

## What It Does

- Loads the current cached in-game bulletin feed.
- Supports browsing by tab: `all`, `events`, `updates`, and `news`.
- Lets you jump directly to an event from autocomplete or page through the full catalog.

## Additional Notes

- This command does not require a linked account.
- Event results are localized using the user context when available.
