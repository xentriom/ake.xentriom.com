## Enka command

Reserve a future Enka.Network profile lookup path.

## Usage

```text
/enka
/enka uid:<uid>
```

## Parameters

- `uid` (optional)
  Uses a manual UID instead of falling back to your first linked account.

## What It Does

- Accepts a manual UID or falls back to the first linked account when no UID is supplied.
- Currently returns a warning message indicating that Enka integration is waiting on API stability.

## Additional Notes

- This command is present in the slash surface, but the current implementation is still a placeholder.
