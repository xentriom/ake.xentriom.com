## Current Scope

Endvoyant currently supports **Arknights: Endfield**.

This is not a multi-game utility bot right now. The linked-account flow, profile commands, event
feed, attendance actions, and redemption flow in the current source are all built around Endfield
and SKPort-backed account data.

## What Is Covered

- Account linking through [`/add account`](/docs/add)
- Manual and automated attendance
- Profile, regional development, and region exploration views
- Owned-operator browsing and character detail views
- Redemption-code attempts
- Public in-game bulletin browsing through [`/events`](/docs/events)

## What Is Not Covered

- Other Hypergryph or Gryphline games
- Cross-game account linking
- A production-ready Enka.Network integration
- Official first-party support from the game publisher

## Looking Ahead

If Endvoyant expands beyond Endfield, this page should be treated as the source of truth for game
coverage. For the current build, assume all commands and guides are Endfield-specific unless a page
explicitly says otherwise.
